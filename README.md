## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on different platforms:
- **Web**: Press `w` or run `npm run web`
- **Android**: Press `a` or run `npm run android`
- **iOS**: Press `i` or run `npm run ios`

## Modules

The app includes the following modules:

1. **My Customer** - Manage customer database
2. **Delivery Boy** - Track delivery personnel
3. **Daily Sell** - Record daily sales
4. **New Daily Sell** - Enhanced sales recording
5. **Create Bill** - Generate customer bills
6. **Report** - View various reports
7. **Products** - Manage product inventory
8. **Message** - Send messages to customers
9. **Received Payment** - Track payments
10. **Dispute Request List** - Handle disputes
11. **Archive** - Access archived data
12. **What's App** - WhatsApp integration
13. **Milk Report** - Milk quality reports
14. **App Message** - In-app messaging
15. **Group Management** - Manage customer groups
16. **Settings** - App configuration

## Color Theme

The app uses a light green color palette:
- Primary: `#90EE90` (Light Green)
- Secondary: `#4DD0E1` (Cyan)
- Accent: `#66BB6A` (Green)
