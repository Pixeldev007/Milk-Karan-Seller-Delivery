# Quick Start Guide - Milk Karan App

## Prerequisites

Make sure you have Node.js installed on your system. You can download it from [nodejs.org](https://nodejs.org/)

## Installation Steps

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm start
   ```

3. **Choose your platform:**
   - Press `w` to open in web browser (Desktop/Mobile responsive)
   - Press `a` to run on Android emulator
   - Press `i` to run on iOS simulator
   - Scan QR code with Expo Go app on your phone

## Running on Web (Recommended for Desktop)

```bash
npm run web
```

This will open the app in your default web browser. The app is fully responsive and works great on desktop!

## Troubleshooting

If you encounter any issues:

1. Clear npm cache: `npm cache clean --force`
2. Delete node_modules: `rm -rf node_modules`
3. Reinstall: `npm install`
4. Restart: `npm start`
